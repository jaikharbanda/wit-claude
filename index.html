This Saturday's prompts

Everything we type in the session, ready to copy. The full interview prompt is in this week's free skill below.

Paste 1 Global instructions · Settings → Cowork → Edit Global Instructions
I usually start my Cowork session by pointing you to my Cowork folder.

Before any and every single task, you must read every file in ABOUT ME/:

- about-me: it's me, who I am, what I love and hate
- anti-ai-writing-style: I hate how Claude writes, unless you write and then audit it against my anti-ai-writing-style file.
- my-company: where I work, my role.

Never read the folders OUTPUTS/ or TEMPLATES/ unless I specifically point you to a file.

Save all deliverables in OUTPUTS/ under a subfolder named after the project.

If the brief is unclear, use AskUserQuestion. Don't fill gaps with filler. Don't over-explain. Deliver the work.
Paste 2 Trim a bloated about-me file

Ask me questions on how to trim effectively until we have the perfect document.
Paste 3 Save your best work as a template

Paste 4 Carry a long session into a fresh one
Summarize everything we've done and decided in this session in a compact brief I can paste into a new session. Include the current state of the work, open questions, and next steps.

Quick comparisons

The two things people always ask in week 1.

Claude vs ChatGPT

They've largely converged: on most everyday tasks the top models are close. The difference now is fit and workflow, not raw power.

Claude leans into
  • Coding and long, complex documents
  • Natural writing and tone
  • Working with your files (Cowork + Skills)
ChatGPT leans into
  • Image generation
  • Voice conversations
  • A bigger plugin and app ecosystem

Plenty of people use both. This group focuses on getting great at Claude.

Browser vs Desktop vs Claude Code
Browser Claude
Quick chat and one-off tasks, from any device.
Use for: fast answers, drafting, ideas.
Desktop (Cowork)
Works with your actual files, alongside you.
Use for: real deliverables and documents.
Claude Code
Build tools and websites, even with no coding background.
Use for: building something (week 4).
